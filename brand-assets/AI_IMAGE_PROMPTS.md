## Tips for Using These Prompts

### For Midjourney:
- Add `--ar X:Y` for aspect ratio
- Use `--v 6` for latest version
- Add `--style raw` for photorealistic
- Add `--style modern` for clean tech aesthetic

### For DALL-E 3:
- Be specific about colors (include hex codes)
- Mention "photorealistic" or "illustration" clearly
- Specify aspect ratio and format
- Request "no text" if needed

### For Stable Diffusion:
- Add negative prompts: "blurry, low quality, distorted, text, watermark"
- Use LoRA models for consistent style
- Adjust CFG scale for prompt adherence
- Multiple passes for refinement

### General Tips:
1. **Always specify brand colors** by hex code
2. **Request "no text"** so you can add your own
3. **Ask for clean backgrounds** for versatility
4. **Specify professional/modern aesthetic**
5. **Mention diversity** in people images
6. **Use consistent aspect ratios** for each type

## Batch Generation Strategy

Generate in this order:
1. **Hero images** (1-3) - Most important
2. **Abstract concepts** (9-11) - Versatile backgrounds
3. **Team/people** (4-6) - Humanize brand
4. **Product mockups** (7-8) - Show solution
5. **Social media** (15-17) - Consistent feed
6. **Supporting assets** (18-32) - As needed

## Post-Processing

After AI generation:
1. **Color correction**: Match exact brand hex codes
2. **Add logo**: Overlay kbyg.ai logo where appropriate
3. **Optimize**: Compress for web (JPG 80-85% quality)
4. **Resize**: Create multiple sizes for different uses
5. **Archive**: Save originals + edited versions
