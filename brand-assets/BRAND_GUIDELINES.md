# kbyg.ai Brand Guidelines

## Brand Identity

### Company Name
**kbyg.ai** - Always lowercase, always with ".ai" extension

### Tagline Options
- "Intelligence Extraction for Revenue Teams"
- "Know Before You Go"
- "Unfair Competitive Advantage"
- "Stop Networking; Start Executing"

### Brand Voice
- **AGGRESSIVE**: Revenue-obsessed, domination-focused
- **Tactical**: Execution-driven, no-nonsense, direct
- **Competitive**: Unfair advantage, weaponized intelligence
- **Uncompromising**: Superior preparation, guaranteed results

## Logo Usage

### The Icon
The neural network icon represents:
- Deep-tier intelligence extraction network
- Command and control operations
- Tactical prospect mapping
- Connected nodes = weaponized relationships

### Logo Variations

1. **Primary Logo** (`logo-primary.svg`)
   - Full horizontal lockup
   - Use on light backgrounds
   - Preferred version for most applications

2. **White Logo** (`logo-white.svg`)
   - For dark backgrounds
   - Maintain gradient in icon

3. **Icon Only** (`icon-only.svg`)
   - Social media profiles
   - Favicons
   - App icons
   - Minimum 32x32px

### Clear Space
Maintain clear space equal to the height of the neural network icon on all sides.

### Minimum Sizes
- **Full logo**: 120px width
- **Icon only**: 32px
- **Print**: 1 inch width minimum

### Don'ts
❌ Don't change colors
❌ Don't rotate or skew
❌ Don't add effects or shadows
❌ Don't use on busy backgrounds
❌ Don't separate wordmark from icon in primary logo
❌ Don't use old versions

## Color System

### Primary Palette

**Command Blue** (#3b82f6)
- Primary actions and CTAs
- Links and interactive elements
- Intelligence extraction features
- Authority and execution indicators

**Extraction Purple** (#8b5cf6)
- Deep-tier intelligence features
- Weaponized opener generation
- Secondary CTAs
- Competitive advantage visualizations

**Tactical Cyan** (#06b6d4)
- Revenue-ready metrics
- Execution tracking
- Domination indicators
- Unfair advantage dashboards

### Usage Rules

1. **Primary blue** should dominate (60% usage)
2. **Neural purple** for accents (30% usage)
3. **Cyber cyan** for highlights (10% usage)
4. Never use all three equally - maintain hierarchy
5. Use gradients sparingly for impact

### Accessibility
All color combinations meet WCAG AA standards.
Test contrast ratios before deployment.

## Typography

### Font Family
**Inter** - Single font family for brand consistency

### Font Weights
- 300 (Light): ".ai" in wordmark
- 400 (Regular): Body text
- 500 (Medium): Subheadings
- 600 (Semibold): Emphasis
- 700 (Bold): Headings
- 800 (Extrabold): "kbyg" in wordmark, major headings

### Brand Wordmark Typography
```
kbyg  = Inter 800 (Extrabold), #111827
.ai   = Inter 300 (Light), #3b82f6
```

This contrast emphasizes the AI nature while keeping the brand strong.

## Voice & Tone

### Voice Characteristics
**What we are:**
- AGGRESSIVE and revenue-obsessed
- Tactical and execution-focused
- Competitive and domination-minded
- Direct and uncompromising

**What we're not:**
- Soft or passive language
- Vague or theoretical concepts
- Apologetic or hesitant
- Casual or unprofessional

### Writing Guidelines

**Do:**
- Use active voice
- Be concise and clear
- Explain technical concepts simply
- Focus on benefits, not just features
- Use "we" and "our" for the company
- Use "you" and "your" for the user

**Don't:**
- Use unnecessary jargon
- Make unsupported claims
- Be condescending
- Use buzzwords without substance

### Example Messaging

**Good:**
> "KBYG.ai Command Center gives you an unfair competitive advantage. Extract every high-value prospect, weaponize your openers, and dominate the room. Stop networking; start executing."

**Bad:**
> "Our platform helps you network more effectively at events and connect with potential customers."
